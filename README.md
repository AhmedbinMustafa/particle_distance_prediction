# particle_distance_prediction
### Overview

This project involves the development of a computational model to predict the transport distance and terminal velocity of wood dust particles. The research focuses on the relationship between physical particle properties and environmental variables to improve industrial safety and environmental monitoring.

The video is the example of the project to be performed 


### What We Did

* **Parameter Analysis**: Analyzed the correlation between particle size, density, and wind speed.

* **Distance Prediction**: Modeled the expected travel distance of wood dust based on varying atmospheric conditions.

* **Velocity Calculation**: Determined terminal velocities for different wood dust morphologies.

### How We Did It

* **Mathematical Modeling**: Applied fluid dynamics principles, including **Stokes Law** and the **Reynolds number**, to define particle behavior.

* **Drag Coefficient Analysis**: Utilized the **Schiller-Naumann equation** to calculate accurate drag coefficients for non-spherical particles.

* **Python Implementation**: Developed the predictive model using **Python**, leveraging libraries such as **NumPy** and **Pandas** for data processing and numerical analysis.
